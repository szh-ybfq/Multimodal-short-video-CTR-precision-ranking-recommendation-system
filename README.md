# Multimodal-short-video-CTR-precision-ranking-recommendation-system
## 多模态短视频CTR精排推荐系统
  针对短视频推荐场景中，通用多模态预训练特征与点击预估目标适配性差、用户长短期动态兴趣建模不精准、新内容冷启动效果弱的行业痛点，设计端到端的多模态精排CTR预估模型，适配核心相关业务场景。
## 数据集：
  MicroLens-100k：西湖大学发布，专为短视频推荐设计，官方已提前提取好文本、图像、视频多模态特征，覆盖用户行为序列、视频标题、封面、评论全维度数据。本实验使用预提取特征（extracted_modality_features.zip）直接实验，省去特征提取繁琐步骤，各模态分别为视频封面（图片）、标题（文本）、内容（视频），具体介绍见[MicroLens](https://github.com/westlake-repl/MicroLens) 数据集链接：[MicroLens-100k](https://recsys.westlake.edu.cn/)
## 数据预处理
  按照用户级时间划分，训练集采用热门负采样，正负比例1:4，验证、测试集采用随机负采样，正负比例为1:20，具体见load_data.py文件。
## 模型架构
- 设计多模态自适应融合模块：基于图像、文本、视频预训练特征，通过可学习门控网络自适应学习模态权重，通过多头自注意力层建模封面、标题、视频内容的关联关系，有效缓解新视频冷启动问题。
- 构建长短期兴趣解耦编码模块：通过门控机制过滤行为噪声提取用户长期稳定偏好，带因果掩码的轻量化Encoder建模用户短期兴趣，结合目标感知的兴趣融合门控，实现用户动态兴趣的精准捕捉。
- 基于MoE架构搭建模态感知专家网络：设计目标专属专家与共享专家，结合低权重对比学习辅助正则，对多模态特征和长短期兴趣特征分别做双向对齐，提升模型泛化能力与排序精度。
- 全链路工程性能优化：通过混合精度训练、多进程数据加载、GPU张量预分配、纯numpy向量化指标计算，将单epoch训练耗时优化至44秒，测试集全量评估速度提升5倍以上，大幅提升模型迭代效率。
